初始遗物

宇宙本源

轴伊的初始遗物，定义了角色的核心玩法。

🛡️

伤害减免

受到的所有伤害减免 50%，让你在战斗中更加从容。

🕳️

黑洞转化

当遗物减免伤害时，获得与减免伤害数值相同层数的黑洞，延迟处理所受的伤害。

⚙️

轴芯召唤

召唤轴芯作为你的伙伴，它能存储一张攻击牌并在回合开始时自动打出。

召唤物

轴芯

你忠实的伙伴，战斗中的核心角色。

🛡️ 伤害转移

轴芯会替你承受来自敌方的伤害，保护你的生命值。合理利用轴芯的血量，在进攻和防守之间找到平衡。

🃏 卡牌存储

轴芯能够存储一张攻击牌，并在你的回合开始时自动打出。选择合适的攻击牌存储，打造强力的连击。

⚡ 增强能力

通过"轴芯好帅"能力，轴芯可以打出两次存储的卡牌。配合高伤害攻击牌，造成毁灭性的输出。

开始游戏

安装方法

  1. 从 GitHub Releases 下载最新版本的模组文件
  2. 解压下载的文件到杀戮尖塔2的 mods 文件夹中
  3. 启动游戏，在角色选择界面你会看到"轴伊"
  4. 选择轴伊，开始你的宇宙冒险之旅！
