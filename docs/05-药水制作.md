## 基础概念

### 药水系统
- 药水是一次性使用的消耗品
- 可以在战斗中或战斗外使用
- 有不同的稀有度和体型

### 药水稀有度
- **Common** - 普通（65%）
- **Uncommon** - 罕见（25%）
- **Rare** - 稀有（10%）

## 药水基类

### 基本结构
```csharp
// 路径：JoiCode/Potions/JoiPotion.cs
using BaseLib.Abstracts;
using MegaCrit.Sts2.Core.Models;
using MegaCrit.Sts2.Core.Entities.Potions;

namespace Joi.Potions;

[Pool(typeof(JoiPotionPool))]
public abstract class JoiPotion : CustomPotionModel
{
    public override string PackedIconPath
    {
        get
        {
            var path = $"{Id.Entry.RemovePrefix().ToLowerInvariant()}.png"
                .PotionImagePath();
            return ResourceLoader.Exists(path)
                ? path
                : "potion.png".PotionImagePath();
        }
    }
}
```

### 关键属性
- `Rarity` - 稀有度
- `Usage` - 使用时机（战斗中/战斗外）
- `TargetType` - 目标类型
- `Body` - 药水体型
- `Owner` - 所有者（Player）

## 药水实现示例

### 1. 攻击药水（AttackPotion）
```csharp
[Pool(typeof(JoiPotionPool))]
public class AttackPotion : JoiPotion
{
    public override PotionRarity Rarity => PotionRarity.Common;
    public override PotionUsage Usage => PotionUsage.InCombat;
    public override TargetType TargetType => TargetType.AnyEnemy;
    public override PotionBody Body => PotionBody.Bolt;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Damage", 10)];

    public override async Task OnUse(
        PlayerChoiceContext choiceContext,
        Creature? target)
    {
        if (target != null)
        {
            await CreatureCmd.Damage(choiceContext,
                [target],
                DynamicVars["Damage"].BaseValue,
                ValueProp.Unpowered,
                Owner.Creature,
                null);
        }
    }
}
```

### 2. 治疗药水（HealPotion）
```csharp
[Pool(typeof(JoiPotionPool))]
public class HealPotion : JoiPotion
{
    public override PotionRarity Rarity => PotionRarity.Common;
    public override PotionUsage Usage => PotionUsage.Anytime;
    public override TargetType TargetType => TargetType.Self;
    public override PotionBody Body => PotionBody.Heart;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Heal", 20)];

    public override async Task OnUse(
        PlayerChoiceContext choiceContext,
        Creature? target)

            Owner.Creature,
```

### 3. 能力药水（PowerPotion）
```csharp
[Pool(typeof(JoiPotionPool))]
public class PowerPotion : JoiPotion
{
    public override PotionRarity Rarity => PotionRarity.Uncommon;
    public override PotionUsage Usage => PotionUsage.InCombat;
```

### 4. AOE 药水（ExplosionPotion）
```csharp
[Pool(typeof(JoiPotionPool))]
public class ExplosionPotion : JoiPotion
{
    public override PotionRarity Rarity => PotionRarity.Rare;
    public override PotionUsage Usage => PotionUsage.InCombat;
    public override TargetType TargetType => TargetType.AllEnemies;
    public override PotionBody Body => PotionBody.Sphere;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Damage", 15)];

    public override async Task OnUse(
        PlayerChoiceContext choiceContext,
        Creature? target)
    {
        var enemies = Owner.Creature.CombatState?.Enemies.ToList() ?? [];
        await CreatureCmd.Damage(choiceContext,
            enemies,
            DynamicVars["Damage"].BaseValue,
            ValueProp.Unpowered,
            Owner.Creature,
            null);
    }
}
```

### 5. 抽牌药水（DrawPotion）

## 最佳实践

1. **稀有度设计**
   - Common: 简单的数值效果
   - Uncommon: 条件触发或中等效果
   - Rare: 强力效果或特殊机制

2. **使用时机**
   - InCombat: 战斗效果（伤害、格挡、能力）
   - OutOfCombat: 非战斗效果（治疗、移除卡牌）
   - Anytime: 通用效果

3. **目标类型**
   - Self: 自身增益
   - AnyEnemy: 单体攻击
   - AllEnemies: AOE 攻击
   - None: 无目标效果

4. **体型选择**
   - 选择与效果匹配的体型
   - 保持视觉一致性
   - 考虑玩家的直觉识别

5. **平衡性**
   - 药水应该强力但不过分
   - 考虑使用时机的限制
   - 测试不同场景下的效果
