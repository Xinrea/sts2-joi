## 基础概念

### 敌人系统
- 敌人继承自 `MonsterModel`
- 使用状态机控制行动
- 通过意图系统显示下一步行动

### 敌人类型
- **普通敌人** - 标准战斗单位
- **精英敌人** - 更强的单位
- **Boss** - 关卡 Boss
- **事件敌人** - 特殊事件中的敌人

## 敌人基类

### 基本结构
```csharp
// 路径：JoiCode/Enemies/CustomEnemy.cs
using BaseLib.Abstracts;
using MegaCrit.Sts2.Core.Models;
using MegaCrit.Sts2.Core.MonsterMoves;
using Godot;

public class CustomEnemy : CustomMonsterModel
{
    // 默认动画状态
    public override string? DefaultMoveState => "idle";

    // 最小初始生命值
    public override int MinInitialHp => 40;

    // 最大初始生命值
    public override int MaxInitialHp => 46;

    // 是否接收战斗钩子
    public override bool ShouldReceiveCombatHooks => true;

    // 创建视觉效果
    public override NCreatureVisuals CreateCustomVisuals()
    {
        Texture2D texture = GD.Load<Texture2D>(
            "res://Joi/images/monsters/custom_enemy.png");
        return NodeFactory<NCreatureVisuals>.CreateFromResource(texture);
    }

    // 生成行动状态机
    protected override MonsterMoveStateMachine GenerateMoveStateMachine()
    {
        // 定义行动状态
        return new MonsterMoveStateMachine([/* 行动列表 */]);
    }
}
```

## 行动状态机

### MonsterMoveStateMachine
状态机控制敌人的行动序列。

### 基本行动（MoveState）
```csharp
protected override MonsterMoveStateMachine GenerateMoveStateMachine()
{
    // 攻击行动
    MoveState attack = new("ATTACK",
        async targets => {
            await CreatureCmd.Damage(null, targets.ToList(),
                8, ValueProp.Move, Creature, null);
        },
        [new AttackIntent(8)]);

    // 防御行动
    MoveState defend = new("DEFEND",
        async targets => {
            await CreatureCmd.Block(null, Creature, 5);
        },
        [new BlockIntent(5)]);

    // 增益行动
    MoveState buff = new("BUFF",
        async targets => {
            await PowerCmd.Apply<StrengthPower>(
                Creature, 2, Creature, null);
        },
        [new BuffIntent()]);

    return new MonsterMoveStateMachine([attack, defend, buff], attack);
}
```

### 行动参数
- **id** - 行动 ID（唯一标识）
- **action** - 行动逻辑（异步方法）
- **intents** - 意图列表（显示给玩家）

## 意图系统

### 常用意图类型

```csharp
new AttackIntent(damage: 8)

```

#### 2. 格挡意图（BlockIntent）

#### 3. 增益意图（BuffIntent）
```csharp
new BuffIntent()
```

#### 4. 减益意图（DebuffIntent）
```csharp
new DebuffIntent()
```

#### 5. 召唤意图（SummonIntent）
```csharp
new SummonIntent()
```

#### 6. 未知意图（UnknownIntent）
```csharp
new UnknownIntent()
```

#### 7. 眩晕意图（StunIntent）
```csharp
new StunIntent()
```

```csharp

// 多段攻击 + 减益

```

## 敌人实现示例

### 1. 简单敌人（Axebot）
```csharp
public class Axebot : CustomMonsterModel
{
    public override int MinInitialHp => 40;
    public override int MaxInitialHp => 46;

    protected override MonsterMoveStateMachine GenerateMoveStateMachine()
    {
        // 启动行动
        MoveState startup = new("STARTUP",
            async targets => {
                await PowerCmd.Apply<StrengthPower>(
                    Creature, 1, Creature, null);
            },
            [new BuffIntent()]);

        // 一连击
        MoveState strike1 = new("STRIKE_1",
            async targets => {
                await CreatureCmd.Damage(null, targets.ToList(),
                    6, ValueProp.Move, Creature, null);
            },
            [new AttackIntent(6)]);

        // 二连击
        MoveState strike2 = new("STRIKE_2",
            async targets => {
                await CreatureCmd.Damage(null, targets.ToList(),
                    4, ValueProp.Move, Creature, null);
                await CreatureCmd.Damage(null, targets.ToList(),
                    4, ValueProp.Move, Creature, null);
            },
            [new AttackIntent(4, 2)]);

        return new MonsterMoveStateMachine(
            [startup, strike1, strike2],
            startup);
    }
}
```

### 2. 复杂敌人（带状态转换）
```csharp
public class ComplexEnemy : CustomMonsterModel
{
    public override int MinInitialHp => 80;
    public override int MaxInitialHp => 90;

    protected override MonsterMoveStateMachine GenerateMoveStateMachine()
    {
        // 第一阶段：防御
        MoveState phase1 = new("PHASE_1",
            async targets => {
                await CreatureCmd.Block(null, Creature, 10);
            },
            [new BlockIntent(10)]);

        // 第二阶段：攻击
        MoveState phase2 = new("PHASE_2",
            async targets => {
                await CreatureCmd.Damage(null, targets.ToList(),
                    15, ValueProp.Move, Creature, null);
            },
            [new AttackIntent(15)]);

        // 第三阶段：狂暴
        MoveState phase3 = new("PHASE_3",
            async targets => {
                await PowerCmd.Apply<StrengthPower>(
                    Creature, 3, Creature, null);
                await CreatureCmd.Damage(null, targets.ToList(),
                    20, ValueProp.Move, Creature, null);
            },
            [new BuffIntent(), new AttackIntent(20)]);

        // 设置状态转换条件
        phase1.AddTransition(phase2, () => Creature.CurrentHp < Creature.MaxHp * 0.7m);
        phase2.AddTransition(phase3, () => Creature.CurrentHp < Creature.MaxHp * 0.3m);

        return new MonsterMoveStateMachine([phase1, phase2, phase3], phase1);
    }
}
```

### 3. 召唤型敌人
```csharp
public class SummonerEnemy : CustomMonsterModel
{
    public override int MinInitialHp => 50;
    public override int MaxInitialHp => 60;

    protected override MonsterMoveStateMachine GenerateMoveStateMachine()
    {
        // 召唤行动
        MoveState summon = new("SUMMON",
            async targets => {
                // 召唤小怪
                var minion = Creature.CombatState.CreateCreature(
                    ModelDb.Monster<MinionEnemy>().ToMutable(),
                    CombatSide.Enemy,
                    null);
                Creature.CombatState.AddCreature(minion);
            },
            [new SummonIntent()]);

        // 攻击行动
        MoveState attack = new("ATTACK",
            async targets => {
                await CreatureCmd.Damage(null, targets.ToList(),
                    10, ValueProp.Move, Creature, null);
            },
            [new AttackIntent(10)]);

        return new MonsterMoveStateMachine([summon, attack], summon);
    }
}
```

## 特殊机制

### 1. 死亡时触发
```csharp
public override async Task OnDeath(PlayerChoiceContext choiceContext)
{
    // 死亡时对所有玩家造成伤害
    var players = Creature.CombatState?.Allies.ToList() ?? [];
    await CreatureCmd.Damage(choiceContext, players,
        10, ValueProp.Unpowered, Creature, null);
}
```

### 2. 回合开始/结束
```csharp
public override async Task OnTurnStart(PlayerChoiceContext choiceContext)
{
    // 回合开始时获得力量
    await PowerCmd.Apply<StrengthPower>(Creature, 1, Creature, null);
}

public override async Task OnTurnEnd(PlayerChoiceContext choiceContext)
{
    // 回合结束时造成伤害
    var players = Creature.CombatState?.Allies.ToList() ?? [];
    await CreatureCmd.Damage(choiceContext, players,
        5, ValueProp.Unpowered, Creature, null);
}
```

### 3. 受到伤害时
```csharp
public override async Task AfterTakingDamage(
    PlayerChoiceContext choiceContext,
    DamageResult damageResult)
{
    // 受到伤害时获得格挡
    await CreatureCmd.Block(choiceContext, Creature, 3);
}
```

## 最佳实践

1. **HP 设计**
   - 普通敌人：30-60 HP
   - 精英敌人：80-150 HP
   - Boss：200-500 HP

2. **行动设计**
   - 简单敌人：2-3 种行动
   - 复杂敌人：4-6 种行动
   - Boss：多阶段行动

3. **意图显示**
   - 清晰显示下一步行动
   - 使用组合意图表示复杂行动
   - 考虑玩家的反应时间

4. **平衡性**
   - 测试不同玩家数量下的难度
   - 考虑与其他敌人的配合
   - 避免过于强力或过于弱小

5. **视觉效果**
   - 使用清晰的图标
   - 与敌人主题协调
   - 考虑动画效果
