## 基础概念

### 卡牌类型（CardType）
- `Attack` - 攻击牌
- `Skill` - 技能牌
- `Power` - 能力牌

### 卡牌稀有度（CardRarity）
- `Basic` - 基础牌（初始卡组）
- `Common` - 普通牌
- `Uncommon` - 罕见牌
- `Rare` - 稀有牌

## 卡牌基类

### 自定义基类结构
```csharp
// 路径：JoiCode/Cards/JoiCard.cs
public abstract class JoiCard(int cost, CardType type, CardRarity rarity, TargetType target) :
    CustomCardModel(cost, type, rarity, target)
{
    // 大卡牌图像路径（1000x760 或 500x380）
    public override string? CustomPortraitPath =>
        $"{Id.Entry.RemovePrefix().ToLowerInvariant()}.png".BigCardImagePath();

    // 小卡牌图像路径（250x190）
    public override string? PortraitPath =>
        $"{Id.Entry.RemovePrefix().ToLowerInvariant()}.png".CardImagePath();

    // 自定义悬停提示
    public override IEnumerable<AbstractHoverTip>? ExtraHoverTips
    {
        get
        {
            var tips = new List<AbstractHoverTip>();
            // 添加关键字提示
            return tips;
        }
    }
}
```

### 关键属性说明
- `cost` - 卡牌费用（-1 表示 X 费用）
- `type` - 卡牌类型（Attack/Skill/Power）
- `rarity` - 稀有度（Basic/Common/Uncommon/Rare）
- `target` - 目标类型（见下节）

## 目标类型

### TargetType 枚举
```csharp
// 原版游戏支持的目标类型
public enum TargetType
{
    None,              // 无目标
    Self,              // 自身
    AnyEnemy,          // 任意敌人
    AllEnemies,        // 所有敌人
    RandomEnemy,       // 随机敌人
    AnyPlayer,         // 任意玩家
    AnyAlly,           // 任意友方（包括召唤物）
    AllAllies,         // 所有友方
    TargetedNoCreature,// 目标但不针对生物
    Osty               // 专属：Osty 召唤物
}
```

### 常用目标类型示例
| 目标类型 | 使用场景 | 示例卡牌 |
|---------|---------|---------|
| `Self` | 自身增益、格挡 | Defend, Teleport |
| `AnyEnemy` | 单体攻击 | Strike, BlackHoleStrike |
| `AllEnemies` | AOE 攻击 | Supernova, CosmicExplosion |
| `AnyAlly` | 友方增益、召唤物交互 | FeedZhouXin |

## 动态变量系统

### 什么是动态变量
动态变量（DynamicVar）用于定义卡牌的可变数值，如伤害、格挡、抽牌数等。这些数值可以被升级、遗物、能力等修改。

### 常用动态变量类型
```csharp
// 伤害变量
new DamageVar(6, ValueProp.Move)

// 格挡变量
new BlockVar(5)

// 抽牌变量
new CardsVar(2)

// 自定义变量
new DynamicVar("Damage", 5)
new DynamicVar("Stacks", 3)
```

### 在卡牌中定义动态变量
```csharp
protected override IEnumerable<DynamicVar> CanonicalVars =>
    [new DamageVar(6, ValueProp.Move)];
```

### 访问动态变量
```csharp
// 通过属性访问
int damage = DynamicVars.Damage.BaseValue;

// 通过索引访问
int stacks = DynamicVars["Stacks"].BaseValue;
```

## 卡牌实现示例

### 1. 基础攻击卡（Strike）
```csharp
[Pool(typeof(JoiCardPool))]
public class Strike : JoiCard
{
    public Strike() : base(1, CardType.Attack, CardRarity.Basic, TargetType.AnyEnemy) { }

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DamageVar(6, ValueProp.Move)];

    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        await CommonActions.CardAttack(this, cardPlay).Execute(choiceContext);
    }

    protected override void OnUpgrade()
    {
        DynamicVars.Damage.UpgradeValueBy(3);
    }
}
```

### 2. 格挡卡（Defend）
```csharp
[Pool(typeof(JoiCardPool))]
public class Defend : JoiCard
{
    public Defend() : base(1, CardType.Skill, CardRarity.Basic, TargetType.Self) { }

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new BlockVar(5)];

    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        await CommonActions.CardBlock(this).Execute(choiceContext);
    }

    protected override void OnUpgrade()
    {
        DynamicVars.Block.UpgradeValueBy(3);
    }
}
```

### 3. 能力卡（GravityField）
```csharp
[Pool(typeof(JoiCardPool))]
public class GravityField : JoiCard
{
    public GravityField() : base(2, CardType.Power, CardRarity.Rare, TargetType.Self) { }

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Damage", 5)];

    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        await CommonActions.ApplySelf<GravityFieldPower>(this, DynamicVars["Damage"].BaseValue);
    }

    protected override void OnUpgrade()
    {
        DynamicVars["Damage"].UpgradeValueBy(2);
    }
}
```

### 4. X 费用卡（CosmicExplosion）
```csharp
[Pool(typeof(JoiCardPool))]
public class CosmicExplosion : JoiCard
{
    public CosmicExplosion() : base(-1, CardType.Attack, CardRarity.Rare, TargetType.AllEnemies) { }

    protected override bool HasEnergyCostX => true;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DamageVar(8, ValueProp.Move)];

    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        var times = ResolveEnergyXValue();
        var damage = DynamicVars.Damage.BaseValue;

        for (int i = 0; i < times; i++)
        {
            await CreatureCmd.Damage(choiceContext,
                CombatState?.Enemies.ToList() ?? [],
                damage, ValueProp.Move, Owner.Creature, this);
        }
    }
}
```

### 5. 交互卡牌（选择卡牌）
```csharp
[Pool(typeof(JoiCardPool))]
public class FeedZhouXin : JoiCard
{
    public FeedZhouXin() : base(0, CardType.Skill, CardRarity.Common, TargetType.AnyAlly) { }

    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        var target = cardPlay.Target;

        // 选择一张手牌
        var card = await CommonActions.SelectSingleCard(this,
            new LocString("cards", "JOI-FEED_ZHOU_XIN.selectionPrompt"),
            choiceContext, PileType.Hand);

        if (card != null && card != this)
        {
            // 消耗选中的卡牌
            await CardCmd.Exhaust(choiceContext, card);

            // 对目标应用效果
            if (target != null)
            {
                await PowerCmd.Apply<StoredCardPower>(target, 1);
            }
        }
    }
}
```

## 升级机制

### OnUpgrade 方法
```csharp
protected override void OnUpgrade()
{
    // 增加伤害
    DynamicVars.Damage.UpgradeValueBy(3);

    // 增加格挡
    DynamicVars.Block.UpgradeValueBy(2);

    // 减少费用
    Cost--;

    // 添加关键字
    Keywords.Add(Keyword.Exhaust);
}
```

### 升级后的文本变化
升级会自动更新卡牌描述中的数值。使用 `{0}`, `{1}` 等占位符引用动态变量。

### 文件路径

### 配置格式
```yaml
JOI-STRIKE:
  name: Strike
  description: Deal {0} damage.
  upgradeDescription: Deal {0} damage.

JOI-DEFEND:
  name: Defend
  description: Gain {0} Block.
  upgradeDescription: Gain {0} Block.

JOI-FEED_ZHOU_XIN:
  name: Feed Zhou Xin
  description: Choose a card from your hand to Exhaust. Store it in Zhou Xin.
  selectionPrompt: Choose a card to feed to Zhou Xin
```

### 占位符说明
- `{0}` - 第一个动态变量
- `{1}` - 第二个动态变量
- 以此类推

## 最佳实践

1. **命名规范**
   - 类名使用 PascalCase
   - 文件名与类名一致
   - 图片文件名使用小写加下划线

2. **费用设计**
   - 基础攻击：1 费
   - 基础格挡：1 费
   - 强力效果：2-3 费
   - X 费用：-1

3. **升级平衡**
   - 伤害/格挡：+2 到 +3
   - 费用：-1（谨慎使用）
   - 添加关键字（如 Innate）

4. **代码组织**
   - 使用 `[Pool(typeof(JoiCardPool))]` 注册
   - 继承自 `JoiCard` 基类
   - 实现 `OnPlay()` 和 `OnUpgrade()`

5. **异步操作**
   - 所有游戏操作都是异步的
   - 使用 `await` 关键字
   - 按顺序执行需要等待的操作
