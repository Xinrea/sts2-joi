## 基础概念

### 遗物系统
- 遗物是永久性的被动效果
- 通过钩子方法响应游戏事件
- 可以修改伤害、格挡、卡牌效果等

### 遗物类型
- **Starter** - 初始遗物
- **Common** - 普通遗物
- **Uncommon** - 罕见遗物
- **Rare** - 稀有遗物
- **Boss** - Boss 遗物
- **Shop** - 商店遗物
- **Event** - 事件遗物

## 遗物基类

### 自定义基类结构
```csharp
// 路径：JoiCode/Relics/JoiRelic.cs
using BaseLib.Abstracts;
using Joi.Extensions;

namespace Joi.Relics;

[Pool(typeof(JoiRelicPool))]
public abstract class JoiRelic : CustomRelicModel
{
    public override string PackedIconPath
    {
        get
        {
            var path = $"{Id.Entry.RemovePrefix().ToLowerInvariant()}.png"
                .RelicImagePath();
            return ResourceLoader.Exists(path)
                ? path
                : "relic.png".RelicImagePath();
        }
    }
}
```

### 关键属性
- `Rarity` - 稀有度
- `Owner` - 所有者（Player）
- `Status` - 状态
- `PackedIconPath` - 图标路径（256 × 256 px）

### 伤害修改钩子
```csharp
// 修改受到的伤害（在 Osty 之前）
public override decimal ModifyHpLostBeforeOsty(
    Creature target,
    decimal amount,
    ValueProp props,
    Creature? dealer,
    CardModel? cardSource)
{
    // 减少 50% 伤害
    if (target == Owner.Creature && dealer != null)
    {
        return amount * 0.5m;
    }
    return amount;
}

// 修改造成的伤害
public override decimal ModifyDamageDealt(
    Creature target,
    decimal amount,
    ValueProp props,
    Creature? dealer,
    CardModel? cardSource)
{
    // 增加 20% 伤害
    if (dealer == Owner.Creature)
    {
        return amount * 1.2m;
    }
    return amount;
}
```

## 遗物实现示例

### 1. 初始遗物（CosmicOriginRelic）
```csharp
[Pool(typeof(JoiRelicPool))]
public class CosmicOriginRelic : JoiRelic
{
    public override RelicRarity Rarity => RelicRarity.Starter;

    // 减少 50% 伤害，并应用黑洞能力
    public override decimal ModifyHpLostBeforeOsty(
        Creature target,
        decimal amount,
        ValueProp props,
        Creature? dealer,
        CardModel? cardSource)
    {
        if (target == Owner.Creature &&
            dealer != null &&
            dealer.Side != target.Side)
        {
            var reduced = amount * 0.5m;
            var blackHoleStacks = (int)Math.Ceiling(amount - reduced);

            if (blackHoleStacks > 0)
            {
                _ = PowerCmd.Apply<BlackHolePower>(
                    target, blackHoleStacks, target, null);
            }
            return reduced;
        }
        return amount;
    }
}
```

### 2. 回合触发遗物（Akabeko）
```csharp
[Pool(typeof(JoiRelicPool))]
public class Akabeko : JoiRelic
{
    public override RelicRarity Rarity => RelicRarity.Common;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Strength", 8)];

    public override async Task AfterSideTurnStart(
        PlayerChoiceContext choiceContext,
        CombatSide side)
    {
        if (side == CombatSide.Player)
        {
            // 第一回合获得力量
            await PowerCmd.Apply<StrengthPower>(
                Owner.Creature,
                DynamicVars["Strength"].BaseValue);

            // 闪烁效果
            await Flash();
        }
    }
}
```

### 3. 卡牌触发遗物
```csharp
[Pool(typeof(JoiRelicPool))]
public class DrawRelic : JoiRelic
{
    public override RelicRarity Rarity => RelicRarity.Uncommon;

    public override async Task AfterCardPlayed(
        PlayerChoiceContext choiceContext,
        CardPlay cardPlay)
    {
        // 每打出一张攻击牌，抽一张牌
        if (cardPlay.Card.Type == CardType.Attack)
        {
            await CardCmd.Draw(choiceContext, Owner, 1);
            await Flash();
        }
    }
}
```

### 4. 伤害修改遗物
```csharp
[Pool(typeof(JoiRelicPool))]
public class DamageBoostRelic : JoiRelic
{
    public override RelicRarity Rarity => RelicRarity.Rare;

    protected override IEnumerable<DynamicVar> CanonicalVars =>
        [new DynamicVar("Percent", 20)];

    public override decimal ModifyDamageDealt(
        Creature target,
        decimal amount,
        ValueProp props,
        Creature? dealer,
        CardModel? cardSource)
    {
        if (dealer == Owner.Creature)
        {
            var percent = DynamicVars["Percent"].BaseValue / 100m;
            return amount * (1 + percent);
        }
        return amount;
    }
}
```

### 5. 战斗开始遗物
```csharp
[Pool(typeof(JoiRelicPool))]
public class CombatStartRelic : JoiRelic
{
    public override RelicRarity Rarity => RelicRarity.Uncommon;

    public override async Task OnCombatStart(
        PlayerChoiceContext choiceContext)
    {
        // 战斗开始时获得格挡
        await CreatureCmd.Block(choiceContext, Owner.Creature, 10);
        await Flash();
    }
}
```

## 本地化配置

### 文件路径
```
Joi/localization/en/relics.yaml
Joi/localization/zh/relics.yaml
```

### 配置格式
```yaml
JOI-COSMIC_ORIGIN_RELIC:
  name: Cosmic Origin
  description: Reduce incoming damage by 50%. Apply Black Hole stacks equal to the reduced amount.

JOI-AKABEKO:
  name: Akabeko
  description: At the start of your first turn, gain {0} Strength.

JOI-DRAW_RELIC:
  name: Card Draw Relic
  description: Whenever you play an Attack card, draw 1 card.
```

### 占位符说明
- `{0}` - 第一个动态变量
- `{1}` - 第二个动态变量

## 常用钩子方法总结

| 钩子方法 | 触发时机 | 用途 |
|---------|---------|------|
| `AfterObtained()` | 获得遗物时 | 初始化 |
| `AfterRemoved()` | 移除遗物时 | 清理 |
| `OnCombatStart()` | 战斗开始时 | 战斗初始化 |
| `AfterSideTurnStart()` | 回合开始时 | 回合触发效果 |
| `AfterTurnEnd()` | 回合结束时 | 回合结束效果 |
| `AfterCardPlayed()` | 卡牌打出后 | 卡牌触发效果 |
| `AfterCardDrawn()` | 抽牌后 | 抽牌触发效果 |
| `ModifyHpLostBeforeOsty()` | 受到伤害前 | 伤害减免 |
| `ModifyDamageDealt()` | 造成伤害时 | 伤害增强 |

## 最佳实践

1. **效果设计**
   - Common: 简单的数值增强
   - Uncommon: 条件触发效果
   - Rare: 强力的机制改变

2. **平衡性**
   - 避免过于强力的效果
   - 考虑与其他遗物的配合
   - 测试不同场景下的表现

3. **视觉反馈**
   - 使用 `Flash()` 提示触发
   - 图标要清晰易识别
   - 尺寸：256 × 256 px

4. **代码组织**
   - 使用 `[Pool(typeof(JoiRelicPool))]` 注册
   - 继承自 `JoiRelic` 基类
   - 实现必要的钩子方法

5. **性能考虑**
   - 避免在钩子中执行复杂计算
   - 缓存常用的值
   - 使用异步操作
